# Clean Input Tool

Kleine Desktop-App mit PySide6/Qt zum Sperren von Eingaben und für weitere modulare Utility-Funktionen.

## Setup

```bash
python -m venv .venv
```

Windows PowerShell:
```powershell
.venv\Scripts\Activate.ps1
```

```bash
python -m pip install -r requirements.txt
python -m useful_utelities_collection.main
```
